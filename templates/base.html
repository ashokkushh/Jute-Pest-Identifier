Description

This Jute Pest Identifier uses Transfer Learning bases EfficientNetB0 model to accurately predict and classify jute pests. Our model, trained on a comprehensive dataset of 6,443 images spanning 17 distinct classes of jute pests, with the accuracy of 98% on the "Trained Dataset", ensuring robust and precise identification. Please note that the "Not-Dog" case is not yet supported in this version of the application.
